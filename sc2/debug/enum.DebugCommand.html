A debug command for the game.

Variants

Text(DebugText)

Shows debug text in the game instance.

Line(DebugLine)

Shows a debug line in the game from p1 to p2.

Aabb(DebugAabb)

Shows a debug axis-aligned bounding box in the game.

Sphere(DebugSphere)

Shows a debug sphere in the game.
